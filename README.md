# Campus Life Super App

## Project Description
Campus Life Super App is a multi-page responsive website designed to help students stay connected with campus life. It provides information about events, student resources, and contact tools in one easy-to-use platform.

## Purpose
The purpose of this project is to create a helpful student-focused web application using modern front-end web development tools.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- External API (Open-Meteo Weather API)
- GitHub Pages

## Features
- Multi-page navigation
- Responsive mobile-friendly design
- Bootstrap components
- Interactive JavaScript alerts
- Live weather data from an API
- Contact form layout

## Setup Instructions
1. Download or clone the repository
2. Open files in VS Code
3. Launch `index.html` in a browser

## Deployment
Hosted using GitHub Pages.
